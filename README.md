# Welcome to JacoMat Pro

## Project info

**URL**: https://ahmedbensalah8603.github.io/Jaco-Mat/

## What is this website?

JacoMat Pro is a website that solves systems of linear equations using the iterative Jacobi method. Input your matrix manually or upload a CSV file for instant calculations with beautiful visualizations.


## How to run the project locally?

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone https://github.com/AhmedBensalah8603/Jaco-Mat.git

# Step 2: Navigate to the project directory.
cd Jaco-Mat

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
