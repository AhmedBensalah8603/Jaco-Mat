# Jaco-Mat

Jaco-Mat est un site web dédié à la méthode de Jacobi et aux matrices. Il permet de résoudre des systèmes d'équations linéaires en utilisant la méthode de Jacobi avec un nombre maximal d'itérations ou une tolérance (epsilon).

## Fonctionnalités
- Résolution des systèmes linéaires avec la méthode de Jacobi.
- Utilisation d'une tolérance (epsilon) pour la convergence.
- Définition d'un nombre maximal d'itérations.
- Vérification de la dominance diagonale de la matrice.
- Affichage des résultats avec le nombre d'itérations et la complexité.

## Algorithmes
Les algorithmes utilisés pour la méthode de Jacobi sont implémentés dans le fichier `jacobiSolver.js`.

## Installation et Utilisation
1. Cloner le dépôt :
   ```bash
   git clone https://github.com/AhmedBensalah8603/Jaco-Mat.git
   ```
2. Installer les dépendances (si nécessaire) :
   ```bash
   npm install
   ```
3. Lancer le projet :
   ```bash
   npm start
   ```
